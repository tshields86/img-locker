== Photo Locker

Photo Locker is a photo management application developed using Ruby on Rails. It incorporates email activation for users when they sign up and uses the transactional email services provider SendGrid to send emails in production. Devise conformable is utilized to build this functionality. The application also uses the Stripe API to accept payment from users to signup for different plans, built from a custom CC form. Amazon S3 bucket handles photo storage in production. Twitter Bootstrap and Sass are utilized for styling.
